# Mapping Saints frontend

This is the frontend for the Mapping Saints portal @ https://saints.dh.gu.se/ that uses the REST-Api from the [django backend for saints](https://github.com/gu-gridh/saints).

## Project Setup

```bash
nvm use 22
npm install
```

### Compile and Hot-Reload for Development

`npm run dev`

### Type-Check, Compile and Minify for Production

`npm run build`
